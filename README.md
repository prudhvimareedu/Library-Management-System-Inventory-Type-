# Library-Management-System-Inventory-Type-
The Library Management System is a console-based application developed using C++. It helps manage book inventory, issue and return operations, and maintains records using file handling for persistent storage.


Features

Add new books to inventory

Issue books to users

Return issued books

View all books with availability status

File-based data storage (books.txt)

Menu-driven user interface
